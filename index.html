Búsqueda Semántica

¿Qué es la búsqueda semántica? A diferencia de la búsqueda por palabras clave, la búsqueda semántica convierte tu consulta en un vector de 384 dimensiones mediante el modelo all-MiniLM-L6-v2. Luego calcula la similitud coseno entre ese vector y los chunks almacenados en MongoDB Atlas. Un score de 1.0 indica identidad perfecta, 0.7+ indica alta relevancia semántica.

Evaluaciones RAGAS BONUS

Métricas RAGAS:
  • Faithfulness: ¿La respuesta es fiel al contexto recuperado? (0–1)
  • Answer Relevancy: ¿La respuesta es relevante para la pregunta? (0–1)
  • Context Recall: ¿El contexto recuperado cubre la respuesta correcta? (0–1)
Las evaluaciones actuales son proxy de similitud coseno (no evaluaciones RAGAS completas con LLM como juez). Para RAGAS completo: pip install ragas.

Implementación RAGAS Completa

from ragas import evaluate
from ragas.metrics import faithfulness, answer_relevancy, context_recall

result = evaluate(
    dataset=eval_dataset,  # HuggingFace Dataset con columnas:
                           # question, answer, contexts, ground_truth
    metrics=[
        faithfulness,       # ¿respuesta fiel al contexto?
        answer_relevancy,   # ¿respuesta relevante a la pregunta?
        context_recall,     # ¿contexto cubre la respuesta correcta?
    ]
)

print(result.to_pandas())
